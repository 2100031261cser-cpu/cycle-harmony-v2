CYCLE PHASE CHECKER - BACKEND LOGIC
================================================================================

OVERVIEW:
The Cycle Phase Checker calculates which phase of the menstrual cycle a user is currently in based on their last period date. This helps recommend the appropriate seed cycling laddus (Phase I or Phase II).

MENSTRUAL CYCLE PHASES:
- Phase I (Follicular Phase): Days 1-14 of the cycle
- Phase II (Luteal Phase): Days 15-28 of the cycle
- Average cycle length: 28 days (customizable)

DATABASE MODEL:

API ENDPOINTS:

BACKEND IMPLEMENTATION:

File: routes/cycle.js
```javascript
const express = require('express');
const router = express.Router();

// Utility function to calculate cycle phase
function calculateCyclePhase(lastPeriodDate, cycleLength = 28) {
  // Convert lastPeriodDate to Date object
  const lastPeriod = new Date(lastPeriodDate);
  const today = new Date();
  
  // Reset time to midnight for accurate day calculation
  lastPeriod.setHours(0, 0, 0, 0);
  today.setHours(0, 0, 0, 0);
  
  // Calculate days since last period
  const timeDiff = today.getTime() - lastPeriod.getTime();
  const daysSinceLastPeriod = Math.floor(timeDiff / (1000 * 60 * 60 * 24));
  
  // Calculate current day in cycle (1-based)
  const currentDay = (daysSinceLastPeriod % cycleLength) + 1;
  
  // Determine phase
  let phase;
  let phaseName;
  let phaseDescription;
  let recommendedProduct;
  let phaseColor;
  
  if (currentDay >= 1 && currentDay <= 14) {
    phase = 'Phase I';
    phaseName = 'Follicular Phase';
    phaseDescription = 'This is the time when your body prepares for ovulation. Estrogen levels rise during this phase.';
    recommendedProduct = 'Phase I Laddus';
    phaseColor = '#4CAF50'; // Green
  } else {
    phase = 'Phase II';
    phaseName = 'Luteal Phase';
    phaseDescription = 'This is the time after ovulation when your body prepares for menstruation. Progesterone levels rise during this phase.';
    recommendedProduct = 'Phase II Laddus';
    phaseColor = '#FFB6C1'; // Pink
  }
  
  // Calculate days until next period
  const daysUntilNextPeriod = cycleLength - currentDay;
  
  // Calculate next period date
  const nextPeriodDate = new Date(lastPeriod);
  nextPeriodDate.setDate(lastPeriod.getDate() + cycleLength);
  
  return {
    currentDay,
    phase,
    phaseName,
    phaseDescription,
    recommendedProduct,
    phaseColor,
    cycleLength,
    daysUntilNextPeriod,
    nextPeriodDate: nextPeriodDate.toISOString().split('T')[0],
    lastPeriodDate: lastPeriod.toISOString().split('T')[0]
  };
}

// POST /api/cycle/calculate
// Calculate current cycle phase
router.post('/calculate', async (req, res) => {
  try {
    const { lastPeriodDate, cycleLength } = req.body;
    
    // Validation
    if (!lastPeriodDate) {
      return res.status(400).json({ 
        error: 'Last period date is required' 
      });
    }
    
    // Validate date format
    const dateRegex = /^\d{4}-\d{2}-\d{2}$/;
    if (!dateRegex.test(lastPeriodDate)) {
      return res.status(400).json({ 
        error: 'Invalid date format. Use YYYY-MM-DD' 
      });
    }
    
    // Validate cycle length
    const validCycleLength = cycleLength || 28;
    if (validCycleLength < 21 || validCycleLength > 35) {
      return res.status(400).json({ 
        error: 'Cycle length must be between 21 and 35 days' 
      });
    }
    
    // Calculate phase
    const result = calculateCyclePhase(lastPeriodDate, validCycleLength);
    
    res.json({
      success: true,
      data: result
    });
    
  } catch (error) {
    console.error('Error calculating cycle phase:', error);
    res.status(500).json({ 
      error: 'Failed to calculate cycle phase',
      message: error.message 
    });
  }
});

// POST /api/cycle/save (Optional)
// Save cycle data for tracking
router.post('/save', async (req, res) => {
  try {
    const { userId, lastPeriodDate, cycleLength } = req.body;
    
    if (!lastPeriodDate) {
      return res.status(400).json({ 
        error: 'Last period date is required' 
      });
    }
    
    const result = calculateCyclePhase(lastPeriodDate, cycleLength || 28);
    
    // Save to database (if CycleData model is implemented)
    const cycleData = new CycleData({
      userId: userId || 'anonymous',
      lastPeriodDate: new Date(lastPeriodDate),
      cycleLength: cycleLength || 28,
      calculatedPhase: result.phase,
      currentDay: result.currentDay
    });
    
    await cycleData.save();
    
    res.json({
      success: true,
      message: 'Cycle data saved successfully',
      data: result
    });
    
  } catch (error) {
    console.error('Error saving cycle data:', error);
    res.status(500).json({ 
      error: 'Failed to save cycle data',
      message: error.message 
    });
  }
});

module.exports = router;
```

ALGORITHM EXPLANATION:

1. Input: Last period date (YYYY-MM-DD format)
2. Calculate days since last period: (Today - Last Period Date)
3. Calculate current day in cycle: (Days Since Last Period % Cycle Length) + 1
4. Determine phase:
   - If current day is 1-14: Phase I (Follicular)
   - If current day is 15-28: Phase II (Luteal)
5. Calculate next period date: Last Period Date + Cycle Length
6. Calculate days until next period: Cycle Length - Current Day
7. Return all calculated data with recommendations

EDGE CASES HANDLED:

1. Future dates: If last period date is in the future, calculation still works
2. Very old dates: Cycles wrap around using modulo operation
3. Invalid dates: Validation checks for proper date format
4. Custom cycle lengths: Supports 21-35 day cycles
5. Timezone issues: Dates are normalized to midnight for consistency
